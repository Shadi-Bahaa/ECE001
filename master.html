  1. Problem solving skill:

    2. Problem solving skill is essential for creating effecient software applications. In problem solving, you learn how to solve real-life problems (and sometimes pure mathematics problems) using all properties, methodolgies, and libraries of the programming language considering the constraints of time and memory. In addition to the language, learning data structures and algorithms can add a lot to your solution. About resources, codeforces represents the most popular website for solving. For more difficult and interview-like problems, hackerrank and leetcode are preferred.

  2. Large-scale projects:

    3. Large-scale projects are not determined by huge number of codes only but also the variety of tools, technologies, and design methodologies. To create such projects, problem solving training is very helpful but isn't enough; one should learn design patterns and software engineering principles to create a strong project with no or little bugs. Additionally, learning frameworks is time-saving and high-effecient way to implement functions. Resources for studying such topics are listed in the last page.
